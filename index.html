Advanced Fracture
Statistical Analysis

Comprehensive, high-performance tools for static, dynamic, and uncertainty analysis of complex fracture networks.

Documentation
FracSTAT - main.py

Core Capabilities

Static Analyzer

Analyze fracture geometry with tools like Map, Histogram, CDF, Rose, Stereonet, Connectivity, and Intensity tabs.

Dynamic Analyzer

Evaluate dynamic properties using Rose, Mohr, Stereonet, and specialized method tabs.

Uncertainty Analyzer

Perform robust uncertainty quantification with Monte Carlo, LHS, and PCE methods.
