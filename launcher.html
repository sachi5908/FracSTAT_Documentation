FracSTAT Home Page

The launcher serves as the primary entry point and home screen for the entire FracSTAT application. It operates as a single root window that embeds both the Analyzer modules and project settings, allowing the software to behave as one continuous desktop program without opening multiple detached windows.

Application Workflow

Before exploring the user interface, it is helpful to understand where the launcher sits within the broader FracSTAT processing pipeline. The diagram below illustrates the start-to-finish workflow, beginning with data preparation and routing through the GUI into the specific calculation modules.

The general workflow includes:

  • Data Acquisition & Preparation: Start with outcrop photographs (including scale and view direction), digitized fault/fracture traces (preferably in SVG format), or tabulated data (Fault/Frac ID, Strike, Dip, Dip Direction, Length, Aperture).
  • Enter FracSTAT GUI: Use the launcher to import data, scale, and calibrate.
  • Calculations: Choose between the Static Analyzer (for Trace/Segment Maps, Histogram Plots, Orientation Plots, Topology, Intensity, etc.), Dynamic Analyzer (for Stress Inputs, Slip Tendency, Dilatation Tendency, Mohr-Coulomb Plots, etc.), or the Uncertainty Analyzer (for Uncertainty Plots and Maps).
  • Visualization & Export: Export your finalized results as PNG/JPEG images, CSV/Excel spreadsheets, PDFs, or Word documents.

Navigating the Launcher Interface

The Home Window is designed to be user-friendly, organizing project management on the left and analytical modules on the right.

1. The Sidebar (Left Panel)

The left panel is your control center for application navigation and managing project files.

  • Version Type of FracSTAT: A badge below the logo indicates your current running version (e.g., v1.0).
  • Workspace Navigation: Use the sidebar links to switch between Home, Projects, Documentation, Settings, and About pages.
  • Project Information: A dedicated form allows you to easily create a new workspace. You must provide a Project Name, Location / Site, and designate a Save Directory before opening a module.
  • Loaded and Saved Projects: A scrollable "Recent Projects" list allows you to quickly load previous sessions with a single click.

2. The Top Header Bar

The top right corner of the home interface provides quick outbound links:

  • Link to online documentation and GitHub: Direct shortcut buttons are available to open the online docs or visit the project's source code repository.

3. The Module Cards (Main View)

The center of the launcher features three large interactive cards corresponding to the primary FracSTAT modules. Each card highlights the tools available within.

  • Static Module: Focused on geometric fracture characterization. Features include Trace and Segment classification, Histogram Plots, Rose Plots, Connectivity & topology mapping, and Profile Cross-sections.
  • Dynamic Module: Focused on stress-based fracture evaluation. Features include Slip & dilation tendency, Mohr circle stress visualization, susceptibility mapping, and stereonet projections.
  • Uncertainty Module: Dedicated to probabilistic risk evaluation. It encompasses Monte Carlo Simulations (MCS), Latin Hypercube Sampling (LHS), and Polynomial Chaos Expansion (PCE).
  • Click to open Module: At the bottom of each card is a prominent button used to launch that specific environment with your defined project parameters.

Exploring Application Pages

The launcher includes several dedicated pages to help you manage your workspace, access help, and configure global preferences.

  • Projects Page: View and manage all your recent workspaces. You can easily load existing sessions, delete old ones, or quickly create new projects from this menu.
  • Documentation Page: Access comprehensive in-app help. This section includes a quick-start guide, data format requirements, keyboard shortcuts, and detailed explanations of the Static, Dynamic, and Uncertainty Analyzers.
  • Settings Page: Customize your global application preferences[cite: 8]. Here you can configure default plot styling (fonts and colors), default stress parameters, map color scales, and export directories[cite: 8].
  • About Page: Find information about the software's version, release notes, open-source licensing (MIT), and the underlying technology stack powering the application.
