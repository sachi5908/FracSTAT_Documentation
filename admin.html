FracSTAT Docs — Admin Panel

Pick a page below to open it for editing. You'll be able to click directly into any paragraph, heading, table cell, code snippet, or note/tip box to change its text, and hover an image to replace it. Hover any block to reveal move / duplicate / delete controls, and click the small "+" bar between blocks to insert a new paragraph, heading, list, table, code snippet, note box, tip box, or image anywhere on the page — including directly above or below the code snippet area. Tables get their own "+ Row / − Row / + Col / − Col" controls. When you're done, use "Download Page" to export the updated file and upload it back to your site.
